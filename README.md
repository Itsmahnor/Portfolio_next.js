# Portfolio Website

A modern and responsive developer portfolio built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**. It showcases my projects, skills, experience, and includes a simple portfolio assistant to help visitors learn more about me.

## Features

- Responsive design
- Next.js 15 App Router
- Dynamic project detail pages
- Portfolio assistant
- Contact form with validation
- SEO-friendly metadata
- Smooth animations
- Optimized images

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- React Hook Form
- Zod
- Lucide React

## Getting Started

Clone the repository:

```bash
git clone https://github.com/Itsmahnor/Portfolio_next.js.git
```

Go to the project directory:

```bash
cd portfolio-nextjs
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

## Build for Production

```bash
npm run build
npm start
```
